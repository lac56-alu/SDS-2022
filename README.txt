En primer lugar accederemos a la ruta en la que se encuentre alojada la carpeta que contiene nuestra aplicación y abriremos dos terminales aquí. Introduciremos el comando go mod tidy, el cual se encargará de descargar todas las dependencias que contenga nuestra aplicación. También ejecutaremos un npm i, para actualizar las dependencias hacia los paquetes de este proveedor.
Una vez tengamos este paso realizado, continuaremos y pasaremos a la puesta en marcha del servidor. Para ello introducimos el siguiente comando: go run .\main.go srv "pass" , la pass se refiere a especificar una contraseña con la cual encriptamos la información del servidor. En el ejemplo que hemos puesto por ejemplo lo que haría sería encriptar la palabra pass usando SHA512 y almacenaríamos esa contraseña en el servidor.
Para inicializar el cliente, una vez tengamos el servidor corriendo, solamente tendremos que repetir la operativa de abrir una nueva terminal en la ubicación de la aplicación y ejecutar el siguiente comando: go run .\main.go cli
